# QC Chart Data Visualizer v4.0

專為健身器材平面設計師打造的 QC Checklist 自動化生成工具

## 🎯 專案目的

解決平面設計師在製作產品品檢清單時的痛點：
- 將多種格式的 artwork (AI/PDF/DWG/圖片) 整合成 QC checklist
- 標註設計在產品上的位置
- 提供比 AI/PPT 更靈活且易於協作的工具
- **重點：簡單易用的箭頭標註功能**

## ✨ 主要功能

### 1. 箭頭標註增強 ⭐ NEW!
- ✅ **自定義粗細**：0.5px - 8px 可調
- ✅ **顏色選擇**：7 種預設顏色 + 自定義顏色選擇器
- ✅ **樣式選項**：實線 / 虛線
- ✅ **連續折線**：支援多點連接（未來版本）
- ✅ **即時預覽**：拖曳時即時顯示箭頭效果

### 2. 專案歷史管理 💾 NEW!
- ✅ **儲存專案**：完整保存所有頁面、圖片、箭頭設定
- ✅ **載入專案**：隨時重新開啟之前的工作
- ✅ **歷史記錄**：查看所有已儲存的專案
- ✅ **刪除管理**：清理不需要的舊專案
- ⚠️ 使用 LocalStorage，建議定期備份重要專案

### 3. 圖片品質設定 🎨 NEW!
- ✅ **96 DPI**：適合網頁預覽、快速分享
- ✅ **150 DPI**：標準品質，平衡檔案大小與清晰度
- ✅ **300 DPI**：高品質印刷，適合正式文件
- ✅ 導出時可選擇 PDF 或 PPT 格式

### 4. 多頁面支援
- 建立多個頁面處理複雜產品
- 每頁獨立的底圖、素材和箭頭
- 頁面標籤管理

### 5. 資料庫系統
- 按產品類型分類（TM/EP/BIKE/STRENGTH）
- 可創建自定義資料夾
- 搜尋功能快速找到素材

### 6. 檔案支援
- 圖片格式：JPG, PNG, GIF, WebP
- PDF 文件：自動轉換為圖片
- 拖放上傳

### 7. A3 畫布
- 1587px × 1123px (A3 橫向)
- 適合印刷輸出
- 縮放檢視 (20% - 120%)

## 🚀 使用方式

### 快速開始

1. **設定專案資訊**
   - 填寫品牌名稱、產品型號、日期、版本

2. **上傳底圖**
   - 點擊「底圖已鎖定」按鈕解鎖
   - 上傳產品圖片
   - 拖曳調整位置、縮放大小
   - 完成後再次鎖定底圖

3. **添加素材**
   - 上傳 artwork 圖片或 PDF
   - 點擊素材添加到畫布
   - 拖曳調整位置和大小
   - 可存入資料庫供日後使用

4. **繪製箭頭**
   - 點擊「箭頭」工具
   - 設定箭頭樣式（粗細、顏色、實線/虛線）
   - 在畫布上拖曳繪製箭頭
   - 點擊箭頭可選取並刪除

5. **儲存專案**
   - 點擊「儲存專案」按鈕
   - 輸入專案名稱
   - 專案會保存在瀏覽器中

6. **導出文件**
   - 點擊「導出」按鈕
   - 選擇圖片品質 (96/150/300 DPI)
   - 選擇 PDF 或 PPT 格式
   - 等待生成並下載

### 載入舊專案

1. 點擊「歷史」按鈕
2. 從列表中選擇專案
3. 點擊「載入」按鈕
4. 繼續編輯

## 💡 使用技巧

1. **箭頭顏色建議**
   - 紅色：重要標註、錯誤提示
   - 橙色：警告、注意事項
   - 綠色：正確示範
   - 藍色：一般說明
   - 黑色：基本標註

2. **DPI 選擇建議**
   - 96 DPI：內部審查、快速分享
   - 150 DPI：一般工廠文件
   - 300 DPI：正式品檢文件、存檔

3. **專案管理**
   - 定期清理不需要的舊專案
   - 重要專案建議導出 PDF 備份
   - LocalStorage 有容量限制，避免儲存過多大圖

## 🐛 已知限制

1. **LocalStorage 限制**
   - 瀏覽器通常限制 5-10MB
   - 圖片過多或過大可能無法儲存
   - 建議壓縮圖片後上傳

2. **箭頭功能**
   - 目前不支援曲線箭頭
   - 連續折線功能待開發
   - 無法添加文字標註（規劃中）

3. **檔案格式**
   - DWG 格式需先轉換為 PDF 或圖片
   - AI 檔案需先導出為 PDF
